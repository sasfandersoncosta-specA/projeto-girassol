Nosso Compromisso

A Girassol está comprometida em garantir a acessibilidade digital para pessoas com deficiência. Estamos continuamente melhorando a experiência do usuário para todos e aplicando os padrões de acessibilidade relevantes para tornar nossa plataforma inclusiva.

Padrões de Conformidade

Nos esforçamos para que a plataforma Girassol esteja em conformidade com as Diretrizes de Acessibilidade para Conteúdo Web (WCAG) 2.1, nível AA. Estas diretrizes explicam como tornar o conteúdo da web mais acessível para pessoas com uma ampla gama de deficiências.

Recursos de Acessibilidade Implementados

Trabalhamos para incluir os seguintes recursos em nossa plataforma:

  • Navegação por Teclado: A maioria das funcionalidades pode ser acessada utilizando apenas o teclado.
  • Estrutura Semântica: Utilizamos HTML semântico para ajudar tecnologias assistivas (como leitores de tela) a entender a estrutura da página.
  • Alternativas Textuais: Fornecemos texto alternativo para imagens significativas.
  • Contraste de Cores: Buscamos manter um bom contraste entre o texto e o fundo.
  • Design Responsivo: A plataforma se adapta a diferentes tamanhos de tela.
  • Legendas e Rótulos: Campos de formulário possuem rótulos claros.
